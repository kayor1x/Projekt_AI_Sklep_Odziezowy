# User Manual

## Roles

- Guest:
  - Can browse listings
  - Can filter and sort items
  - Cannot create or edit listings
- Authenticated user:
  - Can create listings
  - Can edit and delete only own listings
  - Can manage account settings
- Admin:
  - Can access the admin dashboard
  - Can edit or delete any listing
  - Can create, edit, and delete categories
  - Can view the user list

## How To Use The App

1. Open `/listings` to browse the marketplace.
2. Use the filter sidebar to narrow results by category, price, size, condition, or search text.
3. Register or log in to create your own listing.
4. Use `Sell Item` to open the create form.
5. Fill in title, description, price, category, condition, size, and optionally upload up to 3 images.
6. After saving, the item appears in the public marketplace and in your dashboard.
7. Open `/dashboard` to see your own listings and manage them.
8. Use the delete confirmation page before removing a listing.

## Admin Usage

1. Log in with an admin account.
2. Open `/admin` to see the dashboard.
3. Use `Manage categories` to maintain marketplace categories.
4. Use `Review listings` to moderate all listing content.
5. Open `Users` from the admin area to see registered users and their roles.
